# Research Connect Platform

A student-focused academic research exploration platform that connects students with research opportunities.

## Features

- Animated Hero component showcasing the platform's mission
- Modern UI built with shadcn/ui, Tailwind CSS, and Framer Motion
- Responsive design for all device sizes
- Academic-focused content and messaging

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- shadcn/ui components

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Project Structure

- `/components/ui` - Reusable UI components (following shadcn/ui convention)
- `/pages` - Next.js pages
- `/styles` - Global styles and Tailwind configuration

## Customization

The Hero component can be customized by:

- Updating the `titles` array with different academic-focused adjectives
- Changing the body text to reflect your platform's specific mission
- Modifying button labels and icons to match your user flow
