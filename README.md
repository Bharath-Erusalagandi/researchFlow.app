Project Overview
Research Connect is a web application designed to facilitate meaningful connections between students and academic researchers. The platform streamlines the process of discovering research opportunities and initiating professional communication. By leveraging artificial intelligence, Research Connect assists users in identifying relevant professors based on research interests and formulating precise, professionally structured outreach emails.

Key Features
Intelligent Search: Users can discover professors and research supervisors using natural language queries, filtering by field of study, university, and specific research topics.

AI-Assisted Communication: Integrated tools assist in determining the best approach for contact, including parsing research interests to suggest personalized email content.

Academic Networking: Simplifies the pathway for students to find mentorship and research positions.

Secure Authentication: Robust user management utilizing Supabase and Google OAuth for secure access.

Institutional Integration: Features a comprehensive database of universities and academic departments.

Technical Architecture
The application is built using a modern full-stack architecture:

Frontend Framework: Next.js (React)
Styling System: Tailwind CSS
Authentication & Database: Supabase, Google OAuth
Artificial Intelligence: Integration with Groq SDK and LangChain for language processing tasks
Email Services: Integration with Composio for Gmail capabilities

Getting Started

Prerequisites
Node.js (Recent LTS version)
npm (Node Package Manager)

Installation

Clone the repository to your local machine.

Install the necessary dependencies by running:
npm install
